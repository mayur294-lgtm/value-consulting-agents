# Templates

This directory contains structured templates for inputs and outputs in the Value Consulting system.

## Structure

### `/inputs/`
Contracts defining what data is required for each type of analysis.

**Contents:**
- Discovery input requirements
- Financial data schema
- Transcript structure
- System documentation format
- Stakeholder interview guide

### `/outputs/`
Structured templates for all consulting deliverables.

**Contents:**
- Assessment Report template
- ROI Report template
- Capability Assessment template
- Roadmap template
- Executive Summary template
- Assumptions Register template

## Purpose

Templates ensure:
1. **Consistency** across all outputs
2. **Completeness** of required sections
3. **Quality** through structured format
4. **Efficiency** via reusable structure
5. **Standards compliance** with Value Consulting principles

## Usage

Agents use these templates to:
- Validate input completeness
- Structure their outputs
- Ensure all required sections are present
- Maintain consistent formatting
- Follow best practices

Consultants use these templates to:
- Understand what data to provide
- Know what outputs to expect
- Generate consistent deliverables
- Maintain quality standards
