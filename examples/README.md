# Example Engagements

This directory contains reference examples of complete consulting engagements, including inputs, analysis, and outputs.

## Structure

### `/engagements/`
Complete engagement folders, each containing:
- Raw inputs (transcripts, data files, context)
- Analysis work products
- Final deliverables (assessment, ROI, roadmap)
- Lessons learned

## Purpose

Examples serve as:
1. **Training data** for understanding good consulting work
2. **Reference material** for methodology application
3. **Quality benchmarks** for output standards
4. **Templates** for common engagement patterns

## Example Categories

Future examples will cover:
- Digital transformation assessments
- Platform modernization business cases
- Capability maturity assessments
- Strategic roadmap development
- Quick-win opportunity identification

## What Good Examples Show

Each example demonstrates:
- How to interpret ambiguous inputs
- How to handle missing data
- How to document assumptions
- How to structure analysis
- How to generate executive-ready outputs
- How to maintain defensibility
- How to balance thoroughness with clarity

## Usage

When working on new engagements:
1. Review similar examples for pattern matching
2. Adapt methodology to specific context
3. Reference quality standards
4. Learn from documented lessons
5. Avoid repeating past mistakes
