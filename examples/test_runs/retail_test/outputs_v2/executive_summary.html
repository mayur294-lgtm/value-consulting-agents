What Would Change This Decision?

Decision Becomes "No Go" If:
  1. Current completion rate is already >55% (less headroom)
  2. Servicing volume is <400K annually (smaller opportunity)
  3. FTE cost per hour is <$15 (less savings potential)
  4. Implementation costs exceed $12M (breakeven at risk)
External Risks:
  1. Regulatory changes to eKYC requirements
  2. Core banking modernization conflicts
  3. Organizational change resistance (workarounds prevalent)
