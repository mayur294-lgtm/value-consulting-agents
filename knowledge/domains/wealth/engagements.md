## Case Study 1: Intesa San Paolo Wealth Management (ISPWM)

**Region:** Europe (Luxembourg)
**Engagement Date:** October 2024
**Segment:** Private Banking / Wealth Management
**AUM:** €8.9B
**Clients:** ~800 acquired/year
**Status:** Post-POV Business Case

### Current State

| Metric | Old Model | Target (Digitally Enabled) |
|--------|-----------|----------------------------|
| Clients per CRM | 91 | 155 |
| Avg AUM per Client | €1M | €11M |
| Total AUM/CRM | €153M | €259M |
| Revenue | €93M | €101M |
| Cost-to-Income Ratio | 70% | 68% |
| Digital Adoption | <30% | >80% |
| Customer Churn | >10% | <5% |

### Identified Pain Points

#### Customer Experience
| Journey | Challenge | Root Cause | Impact |
|---------|-----------|------------|--------|
| Onboarding | Manual paper-based process, high rework | Lack of digital collaboration, paper application, manual data entry | Client conversion <75% |
| Servicing | No capability for payments, trades, or corporate actions | Current app view-only, no self-service | Digital adoption <30% |
| Engagement | Lack of relevant content, no personalization | Customer data siloed across applications | Churn >10% |

#### CRM/Advisor Experience
| Issue | Impact |
|-------|--------|
| Excessive manual data entry | 65% time on non-value tasks |
| Documentation scattered across systems | Slow turnaround times |
| No single platform for collaboration | Poor customer experience |

### Backbase Solution Impact

| Metric | Improvement |
|--------|-------------|
| Client Conversion Rate | +95% improvement |
| Account Opening TAT | ~3-4 days |
| NPS | +15 points |
| Non-Value Added Task Reduction | 65% |
| Transaction Volume | +80% increase |

### Financial Impact (5-Year)

| Category | Value | Annual Average |
|----------|-------|----------------|
| **Revenue Generation** | | |
| Improved Prospect Conversion | €10M | €2M/year |
| Higher Transactions | €10M | €2M/year |
| Higher Engagement | €7.5M | €1.5M/year |
| AUM Fees | €3.8M | €764K/year |
| Retention Benefits | €2.2M | €438K/year |
| **Total Revenue** | **€22M** | **€4.4M/year** |
| | | |
| **Cost Reduction** | | |
| Personnel Efficiency | €6.5M | €1.3M/year |
| **Total Cost Savings** | **€6M** | **€1.2M/year** |
| | | |
| **Gross Benefits** | **€28.5M** | **€5.7M/year** |

### ROI Summary

## Case Study 3: Goodbody Stockbrokers

**Region:** Europe (Ireland)
**Engagement Date:** July 2022
**Segment:** Wealth Management / Stockbroking
**Status:** Value Consulting Final Report

### Business Context

**Two Key Segments:**
1. **Personal/Wealth Segment** - High Net Worth
2. **Connect Segment** - Mass Affluent

### Growth Ambitions

| Segment | Target |
|---------|--------|
| Wealth Revenue | +50% |
| Wealth Profitability | 3x |
| Fee Base | 2x |
| Connect Market Share | 6% → 40% by 2025 |
| Connect Customers | 2x in 2 years |

### Current State Pain Points

#### Personal/Wealth Segment

| Metric | Current | Best-in-Class | Gap |
|--------|---------|---------------|-----|
| Paperless Leakage | 56% | 18% | 38pp |
| Telephone Leakage | 33% | 28% | 5pp |
| Online Leakage | 18% | 5% | 13pp |
| Journey Time (Best) | 96 hrs | 24 hrs | 4x |
| Journey Time (Worst) | 7,680 hrs | 84 hrs | 91x |
| Process Efficiency | 0.5% | 28% | 56x |

#### Connect Segment

| Metric | Current | Best-in-Class | Gap |
|--------|---------|---------------|-----|
| Online Leakage | 17% | 5% | 12pp |
| Telephone Leakage | 45% | 15% | 30pp |
| Journey Time (Best) | 35 hrs | 8 hrs | 4x |
| Journey Time (Worst) | 95 hrs | 24 hrs | 4x |

### Journey Analysis

| Stage | Current Time | Issue |
|-------|--------------|-------|
| Fact Finding & FF | 3 days | Manual paper process |
| Application | 21 days | Email-based document submission |
| AML/KYC/DD | 3.5 days | Manual verification |
| Decision + Setup + Suitability | 2.5 days + 1 day post | Rework and 4-eye checks |
| **Total** | **31 days** | **vs. 4 days best-in-class** |

### Avoidable Time per Role

| Activity | Role | Time/App | Annual Impact |
|----------|------|----------|---------------|
| Fact Finding & Financial Solution | WM Exec/PM | 40 min | 298 hrs |
| Application | WM Exec/PM | 20 min | 149 hrs |
| AML/KYC/DD | AOT | 25 min | 187 hrs |
| AML/KYC/DD | WMOT | 10 min | 75 hrs |
| Account Setup | WMOT | 5 min | 37 hrs |
| Letter Printing | IT Ops | 15 min | 112 hrs |
| Letter Postage | Dispatch | 15 min | 112 hrs |
| **Total** | | | **~970 hrs/year** |

### Revenue Leakage Analysis

| Metric | Value |
|--------|-------|
| Customers Lost to Leakage | 59/year (16% of 365 applications) |
| Revenue Lost | €273K/year |
| Breakdown | Personal: 19% × €1,702 + Wealth: 81% × €5,388 |

### Personas Identified

| Persona | Segment | Profile | Key Needs |
|---------|---------|---------|-----------|
| Mark Connor | Connect (Mass Affluent) | 38yo Director, Energy & Utilities | Instant onboarding, easy navigation, digital + human |
| Mike O'Reilly | Wealth (HNW) | 42yo Entrepreneur | Fully digital onboarding, portfolio overview, seamless channels |
